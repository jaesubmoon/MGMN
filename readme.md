## 소개

커져가는 반려동물 시장! 보호자에게는 집 근처 나 와 우리 아이에게 딱 맞는 정보가 필요하다!

반려동물을 키우는 사람들을 위해 사용자의 위치를 기반으로 거주지 근처 시설들을 추천하고 실 사용자들의 후기를 공유하는 플랫폼
